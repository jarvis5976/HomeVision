# **App Name**: HomeSense Dashboard

## Core Features:

- MQTT Broker Connection: Establish a real-time connection to the MQTT broker at '192.168.0.3:1885' to receive sensor and device data.
- Real-time Data Visualization: Display incoming MQTT data streams dynamically, showing current values and status updates from various home devices.
- Configurable Topic Subscriptions: Enable basic configuration of which MQTT topics the application should subscribe to and monitor.
- Intuitive Device Overview: Present a clear dashboard-like view summarising the status and key metrics of connected home components.
- Intelligent Data Anomaly Detection: An AI tool that monitors incoming data streams for unusual patterns or anomalies, alerting the user to potential issues with their home devices or environment.

## Style Guidelines:

- Primary color: A balanced and clear medium blue (#538CC6) to convey a sense of reliability and modern technology.
- Background color: An ultra-light, subtle blue-grey (#F0F2F4) to ensure data visibility and provide a clean visual space.
- Accent color: A vibrant aqua-green (#52E099) for interactive elements and highlight notifications, offering a fresh contrast.
- Headline and body font: 'Inter' (sans-serif) for its modern, objective clarity and readability across various data displays.
- Employ simple, clean line icons to represent various device types and data points, maintaining visual consistency with the app's modern aesthetic.
- Utilize a modular, dashboard-centric layout, allowing for an organized display of multiple data widgets and device statuses.
- Implement subtle animations for data updates and transitions to provide a smooth and engaging user experience without being distracting.
